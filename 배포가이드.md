# 도랭킹 웹사이트 — PC에서 인터넷에 올리기 (완전 초보용)

지금 이 폴더 안에는 진짜 작동하는 웹사이트가 다 만들어져 있어요.
지금은 가짜 데이터(클립 6개)로 돌아가는데, 화면은 100% 진짜예요.
아래 순서대로 하면 인터넷 주소(URL)가 생겨서 폰으로도 접속할 수 있어요.

---

## 1. 준비물 설치 (최초 1번만)

1. **Node.js 설치**: nodejs.org 접속 → 초록 버튼(LTS) 눌러서 다운로드 → 설치 (그냥 계속 "다음" 누르면 됨)
2. **GitHub 가입**: github.com 에서 이메일로 회원가입

## 2. 이 폴더를 GitHub에 올리기

1. github.com 로그인 후 오른쪽 위 **+** 버튼 → **New repository**
2. 이름은 `dorank-site` 라고 입력 → **Create repository**
3. 만들어진 페이지에 보면 "uploading an existing file" 같은 링크가 있어요 → 클릭
4. 이 압축 풀어놓은 `dorank-site` 폴더 안의 파일들을 통째로 끌어다 놓기 (드래그 앤 드롭)
   - ⚠️ `node_modules` 폴더가 있으면 그건 빼고 올리세요 (용량이 커서 안 올라감, 사실 없어도 됨)
5. 아래 **Commit changes** 버튼 클릭

## 3. Vercel로 배포 (여기가 진짜 "인터넷에 올리기")

1. **vercel.com** 접속 → **Sign Up** → "Continue with GitHub" 선택 (방금 만든 깃허브 계정으로 로그인)
2. 로그인되면 **Add New → Project** 클릭
3. 방금 올린 `dorank-site` 저장소가 목록에 보일 거예요 → **Import** 클릭
4. 다른 설정 건드릴 필요 없이 그냥 **Deploy** 버튼 클릭
5. 1~2분 기다리면 "축하합니다" 화면과 함께 `dorank-site-xxxx.vercel.app` 같은 주소가 생겨요

**이 주소를 폰 브라우저에 쳐서 들어가면, 지금까지 만든 대시보드가 그대로 폰에서 열려요.**
북마크 해두거나 홈 화면에 추가해두면 앱처럼 쓸 수 있어요.

## 지금 상태 vs 나중 상태

| | 지금 (배포 직후) | 나중 (VPS 연결 후) |
|---|---|---|
| 소재수집 탭 | 가짜 클립 6개 고정으로 보임 | 스크래핑 봇이 실제로 모은 클립 |
| 자동 편집 | 버튼 누르면 "완료됐다고 치는" 시뮬레이션 | 진짜 영상 파일 생성 |
| 발행 | 그냥 상태만 "업로드됨"으로 바뀜 | 진짜 유튜브에 업로드 |

지금 이 상태로도 화면 구경하고 흐름 손에 익히는 용도로는 충분해요.
VPS 서버 준비되면 `app/api/` 폴더 안 파일 3개에 있는 "🔌 VPS 백엔드 연결 지점" 주석 부분만
바꿔주면 진짜로 돌아가게 됩니다 (이 부분은 그때 제가 같이 해드릴게요).
